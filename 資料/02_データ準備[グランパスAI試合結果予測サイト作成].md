# <a name="_gb7g16woh9uk"></a>目的
モデルの作成に必要なデータを取得する

# <a name="_yxj09mcfv5do"></a>注意
- データの取得元は基本的にJ1リーグのみとする
  - J2リーグでの結果は、あまり参考にならないという定性の感覚があるため
- ここで、問題になるのは、J2リーグからJ1リーグに昇格したチームの扱いである
  - 本モデルでは、昇格チームを示すフラグを特徴量とし、チームに関する特徴量の初期値は前年の降格チーム平均値を使用するようにする
# <a name="_hjv969g3mfiu"></a>特徴量リスト
データ取得期間：コロナ後から取得する。2021年シーズン〜2025年シーズン

表は2021年のデータの取得方法を例として記述してある
